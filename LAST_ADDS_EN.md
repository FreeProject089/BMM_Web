# Last Adds - BMM Web

This document summarizes the latest features and improvements added to the Better Mod.Manager website.

## New Features

### Progress Tracker (Development Tracking)
- Progress Modal: Integration of a "Progress" button opening an interactive modal.
- Dynamic Visualization: Circular display of global progress and categorized task list (Complete, In Progress, Testing, Planned).
- Flexible Sourcing: Ability to configure a remote data source (GitHub JSON) or local source via config.json.
- i18n Support: Full translation in French and English.

### Update Management
- TBA Status: Handling of "To Be Announced" status for future update dates.
- Dynamic Version Title: Dynamic display of version titles in countdowns.

### Community & Links
- Reddit Integration: Added link to the official Reddit community in the Credits section.
