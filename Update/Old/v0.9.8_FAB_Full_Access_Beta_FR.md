# Sortie : v0.9.8FAB (Bêta Accès Total)

Nous sommes ravis d'annoncer la transition de la Private Test Build (PTB) vers la **Bêta Accès Total (Full Access Beta)** ! Cette version, **0.9.8FAB**, représente une étape majeure pour Better Mod Manager, avec une forte orientation vers le partage et les outils communautaires.

## Nouveautés de la 0.9.8 FAB

### Dépôt Serveur (Mode Serveur)
Le mode serveur très attendu est désormais pleinement opérationnel.
- **Serveur HTTP Intégré** : Hébergez vos profils directement depuis votre PC avec zéro dépendance externe. Cliquez simplement sur "Démarrer l'Hôte Local".
- **Synchronisation Intelligente** : Les clients récupèrent un manifeste repo.json signé et ne téléchargent que les fichiers manquants ou modifiés.
- **Vérification de Sécurité** : Chaque fichier téléchargé est vérifié via son hachage cryptographique SHA-256 avant l'installation.

### Support de l'Import OMM (Open Mod Manager)
- Vous pouvez désormais importer directement vos structures et mods depuis Open Mod Manager (OMM). Better Mod Manager détectera automatiquement les dépôts compatibles et les adaptera instantanément à notre moteur de Copie Physique Intelligente.

### Documentation Visuelle Interactive
### Optimisation de l'Interface (Bibliothèque)
- **Expérience Unifiée** : L'état "Aucun profil" de la Bibliothèque est désormais strictement identique à celui de l'onglet Profils, assurant une cohérence visuelle sur l'ensemble de l'application.


*(Note : Des fonctionnalités telles que le limiteur d'E/S Disque, le Gestionnaire de Stockage et le tableau de bord des performances étaient déjà présentes dans les versions précédentes et ont continué d'être affinées).*

### Passage de licence Apache 2.0 à GPL 3.0
Nous avons décidé de changer la licence de Better Mod Manager de Apache 2.0 à GPL 3.0 
As noté que l'on repassera sur une licence plus permissive à un stade plus avancé du projet.
