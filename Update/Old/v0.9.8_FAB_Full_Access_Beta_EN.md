# Release: v0.9.8FAB (Full Access Beta)

We are thrilled to announce the transition from Private Test Build (PTB) to **Full Access Beta**! This version, **0.9.8FAB**, represents a massive milestone for Better Mod Manager, heavily focusing on sharing and community tools.

## What's New in 0.9.8 FAB?

### Server Repository (Server Mode)
The highly anticipated Server Mode is fully operational.
- **Integrated HTTP Server**: Host your profiles directly from your PC with zero external dependencies. Just click "Start Local Host".
- **Smart Synchronization**: Clients fetch a signed repo.json manifest and only download missing or changed files.
- **Security Verification**: Every downloaded file is verified against its SHA-256 cryptographic hash before installation.
- **Whitelist & Ban Management**: Full control over who can access your repository. Authorize or block users via IP or Creator Key.
- **Real-Time Monitoring**: See exactly who is connected and track active downloads in real-time.

### OMM (Open Mod Manager) Import Support
- You can now seamlessly import your existing structures and mods from Open Mod Manager (OMM). Better Mod Manager will auto-detect compatible repositories and adapt them instantly to our Smart Physical Copy engine.

### Interactive Visual Documentation
Documentation is no longer just static text. We've integrated an Interactive Diagram Engine:
- **Live Tech Visuals**: See exactly how Mod Activation, Synchronization, and Backups work.
- **Lightweight Architecture**: New diagram explaining the core performance philosophy and smart physical copy engine.
- **Dynamic Translation**: Diagrams now speak your language. All labels and tooltips are localized.
- **Tasky's Guidance**: Our mascot Tasky is embedded within the diagrams to explain complex steps on hover.

### UI Optimization (Mod Library)
- **Unified Experience**: The "No Profile" state in the Library is now strictly identical to the Profiles tab, ensuring a consistent visual experience across the app.


*(Note: Features like the Disk I/O Limiter, Storage Manager, and Performance Dashboard were already introduced in previous versions and continue to be refined).*

### Transition from the Apache 2.0 license to GPL 3.0
We decided to change the license of Better Mod Manager from Apache 2.0 to GPL 3.0 
Noted that we will return to a more permissive license at a later stage of the project.

## Moving from v0.9.7 PTB
If you were using the PTB, your profiles and settings will migrate seamlessly. We highly recommend checking out the new Mod Architecture diagram in the documentation to see the latest logic improvements.
