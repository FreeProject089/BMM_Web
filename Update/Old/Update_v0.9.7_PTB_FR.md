# Bienvenue dans la Public Test Build !

Vous utilisez une version **pré-release** de Better Mod Manager. Cette version Public Test Build (PTB) est conçue pour les tests communautaires et les retours d'expérience.

## L'objectif du PTB
L'objectif principal du PTB est de :
1.  **Tester de nouvelles fonctionnalités** avant qu'elles n'atteignent la version stable.
2.  **Signaler les bugs** trouvés lors de votre utilisation.
3.  **Donner votre avis** sur les changements de l'interface et l'expérience globale.

## Quoi de neuf dans le PTB v0.9.7

### Performances et Refonte
- **Détails de mod instantanés** : Le panneau de détail des mods charge désormais les informations de base instantanément, avec les vérifications de conflit s'exécutant en arrière-plan.
- **Vérification sélective des conflits** : Réduction spectaculaire des lags lors de l'activation/désactivation des mods grâce à l'optimisation du moteur de résolution des conflits.
- **Adieu les gels système** : Nouveau moteur de transfert par morceaux (chunked transfer) garantissant que votre PC reste fluide même lors du déplacement de gigaoctets de mods.

### Nouveau moteur de performance
- **Limiteur d'E/S disque** : Prenez le contrôle de la vitesse de votre disque. Définissez des limites personnalisées pour les disques de jeu, de mods et de sauvegarde pour garder votre système réactif.
- **Tableau de bord de performance** : Moniteur PiP de qualité professionnelle pour le CPU, la RAM et les E/S disque avec relecture de l'historique et exportation CSV.
- **Gestionnaire de stockage** : Analyse approfondie de vos disques, incluant la détection de lecteurs cloud (Drive, Dropbox, etc.) et réseau.
- **Refonte du code** : Centralisation de la gestion de l'état et modularisation du frontend pour une meilleure stabilité.

### Peaufinage de l'UI
- **Couleurs de priorité dynamiques** : Les labels de priorité (MAX, MED, LOW) ont désormais une intensité de couleur adaptative basée sur leur valeur numérique.
- **Activation par double-clic** : Vous pouvez maintenant double-cliquer sur n'importe quelle carte de mod pour basculer rapidement son état.
- **Versionnage automatisé** : La version et la date de build statique sont désormais automatiquement synchronisées à travers toutes les vues.
- **Badge PTB dynamique** : Le badge de version bleu reflète désormais intelligemment la configuration de l'application.

### Internationalisation dynamique
- **Auto-détection** : L'application détecte maintenant automatiquement les nouveaux fichiers de langue ajoutés au dossier Lang.
- **Sélecteur de langue unifié** : Un menu déroulant unique et propre dans les paramètres avec l'intégration FlagCDN pour des drapeaux de haute qualité.
- **Outils de traduction** : Nouveaux boutons dans les Paramètres pour accéder au guide de traduction et copier le modèle JSON.
